# the-Particle
🎮 ゲームの特徴
階層的な生産システム: Accelerator Mk.1 から Mk.8 までの加速器を建設し、自動的に粒子を生産します。
多段プレステージ（強くてニューゲーム）システム:
Linac (ライナック): 加速器をリセットして、全体的な生産倍率を獲得します。
Shift (シフト): すべてを失う代わりに、Linacのベース倍率を底上げし、爆発的な成長を可能にします。
Big Crunch (ビッグ・クランチ): 粒子の数が宇宙の限界（1.79e308）に達した時、宇宙は崩壊し、Infinity Pointを獲得します。
SF演出: ゲームの進行度に応じて画面にグリッチノイズが走り、宇宙の限界が近いことを視覚的に表現します。
オートバイヤー: ゲームが進むと自動購入機能がアンロックされ、放置プレイが加速します。
セーブ機能: ブラウザのローカルストレージへの自動保存に加え、テキスト形式でのデータ書き出し/読み込みに対応。
